#### history.pushState(arg1, arg2, arg3)
- arg1: 可以做 `JSON.stringify()`的物件, 也可為 Date, RegExp, ImageData, FileList, File, Blob
- arg2: (optional) 標題(title), 可以用來作為 <Back>選單
- arg3: (optional) 網址(URL), 它會被顯示為目前狀態的位址
